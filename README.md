# Assignment1 - Animals and their sounds
1a) Show picture of a cat with heading CAT.  When you click anywhere on the picture of the cat, you should play an audio file giving "meow".    

1b) Show picture of a cat and a dog, next to each other.  Below the cat, CAT should be written, Below the dog, DOG should be written. If you click on the cat, "meow" audio to be played,  if you click on the dog, "bow bow" audio to be played.    

1c) Show picture of 9 animals in a 3*3 gallery view.  The following animal pictures may be shown - Cat, Dog, Horse, Elephant, Cow, Tiger, Goat, Frog
Above this, show a search box.  If you type name of any animal, only that animal's picture should come up.   Suppose, you type "C", then picture of Cat and Cow should come up.  Now, if you click on the animal, the corresponding sound should be heard.   

1d) Create a menu (... 3 dots), choose a category - eg. Birds.  Now, you should be a able to see 9 pictures of Birds, but all of them coming from a remote server hosted in the cloud using REST API.  The backend URL will be provided here as "http://linode.com/birds.."
The app should be modified to include all the categories available in this REST API.
